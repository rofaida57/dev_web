الكود الوظيفة
النصوص والعناوين
<h1> - <h6>عناوين النصوص بمستويات مختلفة
<p>فقرة نصية
<br>فاصل سطر جديد
<hr>خط أفقي لفصل المحتوى
<strong>نص بارز (غامق)
<em>نص مائل
<mark>تمييز نص بخلفية
<small>تصغير حجم النص
<del>إظهار النص كأنه محذوف
<ins>إظهار النص كأنه مضاف
<sub>نص سفلي
<sup>نص علوي
الروابط والصور
<a href="URL">رابط تشعبي
<img src="URL" alt="نص بديل">إدراج صورة
القوائم
<ul>قائمة غير مرتبة
<ol>قائمة مرتبة
<li>عنصر داخل قائمة
<dl>قائمة وصفية
<dt>العنصر الرئيسي في القائمة الوصفية
<dd>وصف العنصر في القائمة الوصفية
الجداول
<table>إنشاء جدول
<tr>صف داخل الجدول
<td>خلية بيانات
<th>خلية عنوان
<thead>رأس الجدول
<tbody>جسم الجدول
<tfoot>تذييل الجدول
<colgroup>تحديد مجموعة أعمدة
<col>تحديد عمود داخل <colgroup>
النماذج والإدخالات
<form>إنشاء نموذج إدخال بيانات
<input type="text">إدخال نصي
<input type="password">إدخال كلمة مرور
<input type="email">إدخال بريد إلكتروني
<input type="number">إدخال رقم
<input type="date">إدخال تاريخ
<input type="file">رفع ملف
<input type="checkbox">مربع اختيار
<input type="radio">زر اختيار
<textarea>حقل نصي متعدد الأسطر
<select>قائمة منسدلة
<option>خيار داخل القائمة المنسدلة
<button>زر قابل للضغط
هيكلة الصفحة
<header>رأس الصفحة
<footer>تذييل الصفحة
<nav>شريط تنقل
<section>قسم داخل الصفحة
<article>محتوى مستقل
<aside>محتوى جانبي
<main>القسم الرئيسي للصفحة
الوسائط
<audio controls></audio>إدراج ملف صوتي
<video controls></video>إدراج ملف فيديو
<source>مصدر ملف صوتي أو فيديو
<iframe src="URL">إدراج صفحة داخل صفحة
الأكواد والبرمجة
<code>تنسيق النص ككود برمجي
<pre>عرض النص كما هو مع الاحتفاظ بالمسافات
<kbd>إبراز مدخلات لوحة المفاتيح
<samp>إبراز مخرجات البرامج
عناصر متقدمة
<progress>شريط تقدم
<meter>مؤشر قيمة ضمن نطاق
<details>قسم قابل للطي
<summary>عنوان لقسم قابل للطي
عناصر التحكم في العرض
<div>عنصر تقسيم عام
<span>عنصر نصي يستخدم للتنسيق
<bdo dir="rtl">تغيير اتجاه النص
<time>تمثيل التاريخ والوقت
