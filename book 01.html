: HTML اساسيات

الكود الوظيفة
<ul> قائمة غير مرتبة (نقاط)
<ol> قائمة مرتبة (أرقام)
<li> عنصر داخل قائمة (<ul> أو <ol>)
<table> إنشاء جدول
<tr> صف داخل الجدول
<td> خلية داخل الجدول
<th> خلية عنوان في الجدول (تظهر بخط غامق)
<form> إنشاء نموذج إدخال بيانات
<input type="text"> إدخال نصي في النموذج
<input type="password"> إدخال كلمة مرور
<input type="submit"> زر إرسال البيانات
<textarea> حقل نصي متعدد الأسطر
<select> قائمة منسدلة
<option> خيار داخل القائمة المنسدلة
<button> زر قابل للضغط
<div> عنصر تقسيم عام بدون معنى محدد
<span> عنصر نصي يستخدم لتنسيق جزء من النص
<header> رأس الصفحة
<footer> تذييل الصفحة
<nav> قائمة تنقل
<section> قسم داخل الصفحة
<article> مقالة أو محتوى مستقل
<aside> محتوى جانبي
<main> يحدد القسم الرئيسي للمحتوى
<iframe src="URL"> إدراج صفحة داخل صفحة (إطار مضمّن)
