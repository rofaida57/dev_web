جرب البحث بكلمات أخرى

طباعة النصوص

<?php echo "مرحبًا بالعالم!"; ?>
يستخدم لطباعة النصوص على الشاشة

المتغيرات

$age = 25;
تعريف متغير رقمي صحيح
$pi = 3.14;
تعريف متغير رقمي عشري
$name = "أحمد";
تعريف متغير نصي
$isAdult = true;
تعريف متغير منطقي (صواب/خطأ)

العمليات الحسابية

$sum = 10 + 5;
عملية الجمع
$product = 10 * 5;
عملية الضرب
$division = 10 / 2;
عملية القسمة

الشروط

if ($age > 18) { echo "بالغ"; } else { echo "قاصر"; }
شرط إذا كان العمر أكبر من 18

الحلقات

for ($i = 0; $i < 5; $i++) { echo $i; }
حلقة for لتكرار الكود 5 مرات
$age = 20; while ($age < 30) { $age++; }
حلقة while تستمر طالما العمر أقل من 30

المصفوفات

$numbers = array(1, 2, 3, 4, 5);
تعريف مصفوفة أرقام
echo $numbers[0]; // 1
الوصول لعنصر في المصفوفة (الفهرس يبدأ من 0)

الدوال

function add($a, $b) { return $a + $b; }
دالة تجمع رقمين وترجع الناتج

التعامل مع النصوص

$text = "مرحبًا"; echo strlen($text); // طول النص
حساب طول النص

الكائنات والفئات

class Person { public $name; public $age; function __construct($name, $age) { $this->name = $name; $this->age = $age; } }
تعريف فئة Person
$person1 = new Person("محمد", 30);
إنشاء كائن من فئة Person
